# Header Evaluation Rubric

Score: (Passed items / Total items) × 100

## 1. Header Structure (3 items)

- [ ] **Background color**: Deep crimson red (#800000 or similar dark red)
- [ ] **Height/padding**: ~20-25% viewport height with symmetrical vertical padding
- [ ] **Layout**: Content centered horizontally in full-width container

## 2. Emblem/Logo Container (4 items)

- [ ] **Shape**: Horizontal rhombus (diamond) with pointed left/right ends
- [ ] **Border**: Double-line border (thick outer + thin inner), cream colored
- [ ] **Position**: Perfectly centered horizontally and vertically
- [ ] **Size**: 60-70% of total header height

## 3. "SPICY" Text (5 items)

- [ ] **Font**: Decorative vintage serif, engraved/woodcut style
- [ ] **Capitals**: ALL CAPS
- [ ] **Letter spacing**: Noticeable positive tracking (wider)
- [ ] **Color**: Creamy off-white (#F5F5DC or similar)
- [ ] **Position**: Centered, in upper half of rhombus, above chili icon

## 4. Chili Icon (3 items)

- [ ] **Size**: Wide, spanning significant portion of rhombus width
- [ ] **Position**: Centered between "SPICY" and "SPECS" text
- [ ] **Style**: Detailed engraved/woodcut illustration (not flat)

## 5. "SPECS" Text (5 items)

- [ ] **Font**: Identical to "SPICY" (decorative serif, engraved)
- [ ] **Capitals**: ALL CAPS
- [ ] **Letter spacing**: Matches "SPICY" positive tracking
- [ ] **Color**: Same cream as other elements
- [ ] **Position**: Centered, in lower half of rhombus, below chili icon

## 6. Subtitle (5 items)

- [ ] **Content**: Exactly "PATTERN LIBRARY • EST. 2026"
- [ ] **Font size**: Significantly smaller than main text
- [ ] **Weight**: Normal (not bold)
- [ ] **Color**: Same cream as other elements
- [ ] **Position**: Centered, directly below "SPECS", within rhombus bottom

## 7. Ornamental Elements (4 items)

- [ ] **Style**: Victorian/baroque scrollwork and filigree (symmetrical)
- [ ] **Color**: Same cream color as text (#F5F5DC)
- [ ] **Opacity**: Fully opaque and sharply defined
- [ ] **Coverage**: Large scrollwork flanking rhombus L/R + decorative border on top/bottom edges

## 8. Navigation (1 item)

- [ ] **Present**: NO traditional nav menu in header (correct as-is)
